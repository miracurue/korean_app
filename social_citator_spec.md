# Спецификация: Социальный Цитатник и Профили пользователей

## 1. Концепция
Преобразование раздела "Цитаты" в социальное пространство для обмена знаниями. Пользователи не просто пассивно потребляют контент, а активно участвуют в его создании, курировании и распространении.

## 2. Основные компоненты

### 2.1 Глобальная Лента (Social Feed)
- **Типы лент:**
    - **Популярное (Trending):** Цитаты с наибольшим количеством лайков и клонирований за неделю.
    - **Новое (New):** Последние добавленные пользователями цитаты.
    - **Моя подборка:** Цитаты из профилей, на которые пользователь подписан (будущее расширение).
- **Фильтрация:** По тегам (дорамы, песни, уровень ТОПИК, настроение).

### 2.2 Карточка Цитаты (Quote Card)
- **Визуализация:** Текст на корейском, перевод, автор цитаты (герой/дорамы).
- **Метаданные:** Кто добавил цитату (ссылка на профиль контрибьютора).
- **Интерактив:**
    - **Лайк:** Повышение рейтинга цитаты.
    - **Клон (Забрать себе):** Копирование цитаты в личную базу пользователя для изучения.
    - **Ридер:** Кнопка быстрого перехода в *Universal Reader* для детального разбора.
    - **Поделиться:** Генерация эстетичного изображения для соцсетей.

### 2.3 Публичный Профиль (User Profile)
- **Информация:** Аватар, никнейм, краткое био, уровень владения языком.
- **Статистика (Карма):** Количество добавленных цитат, общее число полученных лайков, количество "учеников" (тех, кто клонировал его контент).
- **Коллекции:**
    - **Общедоступные:** Созданные пользователем подборки для всех.
    - **Приватные:** Те, что пользователь "склонировал" у других или создал для себя.

### 2.4 Механика "Clone to Study" (Клонирование)
Это ключевой элемент образовательного процесса. 
1. Пользователь видит интересную фразу в ленте другого юзера.
2. Нажимает "Клон".
3. Цитата копируется в его персональный список.
4. Теперь она доступна в его личном Словаре и может быть запущена в Универсальном Ридере для заучивания.

## 3. Технические требования к данным

### Модель Quote (Цитата)
```typescript
{
  id: string;
  originalCreatorId: string; // Кто первый добавил в систему
  currentOwnerId: string;    // Кому принадлежит эта копия
  isPublic: boolean;         // Видна ли в глобальной ленте
  textKor: string;           // Текст на корейском
  textRus: string;           // Перевод на русский
  audioUrl?: string;         // Ссылка на TTS или оригинальный звук
  source: string;            // Дорама / Книга / Песня
  likesCount: number;
  clonesCount: number;
  tags: string[];            // ['sad', 'drama', 'intermediate']
  createdAt: timestamp;
}
```

### Модель User (Пользователь)
```typescript
{
  id: string;
  username: string;
  displayName: string;
  avatarUrl: string;
  bio: string;
  reputation: number;       // Сумма лайков и клонов его цитат
  achievements: string[];   // Медали: "Полиглот", "Популярный учитель" и т.д.
}
```

## 4. Виральные механики (Viral Growth)
- **Quote-to-Image:** Инструмент на клиенте (Canvas/html2canvas), создающий красивые картинки с цитатой, каллиграфическим шрифтом на фоне размытого кадра из дорамы или абстрактного градиента. Картинка содержит лого платформы и QR-код на профиль автора.

## 5. План реализации (MVP)
1. Базовая страница `Quotes.tsx` с выводом общего списка.
2. Кнопка "Like" и "Clone" с обновлением состояния в локальной базе (mock).
3. Создание страницы `/profile/:username`.
4. Форма добавления новой цитаты с автоматическим TTS (через Google Translate или аналоги).
